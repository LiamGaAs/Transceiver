Utiliser les sous-circuits

Les sous-circuits sont utilisés dans le but d'améliorer la lisibilité d'un schéma, une possibilité très utile dans le cadre de schémas importants, ou dans lesquels un bloc revient plusieurs fois. Dans Qucs, chaque schéma contenant un port de sous-circuit est un sous-circuit. Vous pouvez placer un de ces ports directement en cliquant sur l'icône dans la barre d'outils, ou bien par le menu (Insérer -> Ajouter un port). Après avoir placé tous les ports (deux par exemple), vous enregistrez le schéma (Ctrl-S). Si vous regardez dans la fenêtre de gauche, vous verrez que le nom du fichier est suivi de l'indication « 2-port ». Ce codicille apparaît sur tous les fichiers représentant un sous-circuit. Ouvrez maintenant un schéma dans lequel vous souhaitez utiliser ce sous-circuit. Il vous suffit de sélectionner le fichier sous-circuit, de faire passer la souris dans la fenêtre de travail et de positionner le métasymbole de ce sous-circuit à l'endroit voulu. Complétez votre schéma, et la simulation produira exactement les mêmes résultats que si vous aviez dessiné directement tous les composants du sous-circuit.



Figure 1 - Inclusion d'un sous-circuit dans un schéma

Lorsque vous sélectionnez le métasymbole d'un sous-circuit, vous pouvez en voir le contenu en tapant Ctrl-I (ou son équivalent dans la barre d'outil ou le menu). Pour revenir au schéma principal, tapez Ctrl-H.
